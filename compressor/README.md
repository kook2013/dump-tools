## compressor

Compresses dump files to decrease file size

### Usage
*<X>* required argument *X*
*[Y]* optional argument *Y*

Compress *file_x.dump* and save in *file_x.compressed.dump*

*compressor <file_1.dump> [file_2.dump [file_3.dump [...]]]


### Building

*Dependencies*

- C++11
- Qt5
- qmake
- zlib

*Building*

1. qmake
2. make
